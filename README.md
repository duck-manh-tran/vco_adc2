# VCO based ADC 2nd 
This project implements a second-order VCO-based ADC on skywater 130 nm for Audio Recording. 
## The project includes the following files:
1. `layout`
2. `xschem`
3. `netgen`
4. `matlab`
5. `python`
6. `spice`
7. `synopsys`
